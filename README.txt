Instrucciones para ejecutar el programa ----->> 
 1. Para descargar el proyecto ejecute la siguiente linea en la terminal ---> git clone https://github.com/andresgarciac2/Program2-PSP1.git
 2. Para compilarlo y generar el jar ejecute el siguiente comando ---> mvn clean package install -Dmaven.test.skip=true
 (SI SE EJECUTAN LOS TEST ESTE NO COMPILARA, YA QUE SE HACEN SOBRE ARCHIVOS LOCALES) 
 3. Para ejecutar el jar use el siguiente comando -----> java -cp target/Program2-1.0-SNAPSHOT.jar edu.uniandes.ecos.user.UserInterface
 4. Cuando este se ejecuto debe aparecer un JFrame de java donde usted debe seleccionar el proyecto al cual desea calcularle el tamaño. (SOLO FUNCIONA CON CLASES JAVA)
 5. El resultado de la ejecución aparecera en un JOption y tambien en consola.
